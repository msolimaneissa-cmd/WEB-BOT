- **[GAPS_B] انعدام الاختبارات في الواجهة الأمامية والـ API:** تمت إضافة اختبارات E2E مبنية على Playwright لتغطية صفحات التوثيق (`/login`)، التحكم (`/admin`)، والمسارات الحيوية للـ API (`/api/bot/...`).
