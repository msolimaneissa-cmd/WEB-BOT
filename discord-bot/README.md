# 🤖 Family Legends Discord Bot

بوت ديسكورد متكامل لإدارة السيرفر بالكامل — مجاني ومفتوح المصدر.

## ✨ المميزات

### 🛡️ الإدارة (Moderation)
- `حظر` — حظر عضو من السيرفر
- `طرد` — طرد عضو من السيرفر
- `إسكات` — إسكات عضو مؤقتاً
- `تحذير` — تحذير عضو (يُحفظ في قاعدة البيانات)
- `التحذيرات` — عرض تحذيرات عضو
- `مسح` — حذف رسائل (حتى 100 رسالة)
- `إلغاء_الحظر` — إلغاء حظر عضو

### 🎵 الموسيقى (Music)
- `تشغيل` — تشغيل أغنية أو قائمة تشغيل (YouTube, Spotify, SoundCloud)
- `إيقاف` — إيقاف مؤقت
- `استئناف` — استئناف التشغيل
- `تخطي` — تخطي للأغنية التالية
- `إنهاء` — إيقاف الموسيقى بالكامل
- `قائمة` — عرض قائمة الانتظار
- `الصوت` — تعديل الصوت (1-150%)
- `تكرار` — تكرار (إيقاف / أغنية / قائمة)
- `الآن` — عرض الأغنية الحالية مع شريط التقدم
- `مغادرة` — مغادرة قناة الصوت
- `خلط` — خلط ترتيب القائمة
- `تشغيل_تلقائي` — تشغيل تلقائي لأغاني مشابهة
- `فلتر` — فلاتر صوتية (Bass, Nightcore, 3D, وغيرها)
- `إزالة` — إزالة أغنية من القائمة
- `بحث` — البحث عن أغنية

### 💰 الاقتصاد (Economy)
- `رصيد` — عرض رصيدك أو رصيد عضو آخر
- `يومي` — مكافأة يومية (مع نظام سلسلة الأيام)
- `عمل` — عمل لكسب عملات (كل ساعة)
- `دفع` — دفع عملات لعضو آخر
- `سرقة` — سرقة من عضو آخر (40% نجاح)
- `المتصدرين` — لوحة المتصدرين
- `متجر` — نظام المتجر الكامل

### 🎲 الترفيه (Fun)
- `كرة_الحظ` — سؤال كرة الحظ
- `عملة` — رمي عملة
- `نرد` — رمي نرد
- `ميم` — عرض ميم عشوائي
- `حجر_ورقة_مقص` — لعبة حجر ورقة مقص

### 🛠️ الأدوات (Utility)
- `مساعدة` — عرض كل الأوامر
- `بينغ` — سرعة البوت
- `معلومات_السيرفر` — معلومات السيرفر
- `معلومات_العضو` — معلومات عضو
- `صورة` — صورة بروفايل عضو
- `إعلان` — إرسال إعلان

### ⚙️ التهيئة (Setup)
- `ترحيب` — تعيين قناة الترحيب
- `سجل` — تعيين قناة السجلات
- `رول` — تعيين الرول التلقائي للأعضاء الجدد
- `الحماية` — تشغيل/إيقاف نظام الحماية
- `منع_السبام` — تشغيل/إيقاف منع السبام
- `منع_الروابط` — تشغيل/إيقاف منع الروابط
- `منع_الدعوات` — تشغيل/إيقاف منع دعوات ديسكورد
- `منع_الكابس` — تشغيل/إيقاف منع الحروف الكبيرة
- `منع_الشتائم` — تشغيل/إيقاف منع الشتائم

### 🤖 الحماية التلقائية (Auto-Moderation)
- منع الروابط (anti-link)
- منع دعوات ديسكورد (anti-invite)
- منع الكابس لوك (anti-caps)
- منع السبام (anti-spam) مع عقوبات تلقائية
- منع الشتائم (anti-swear)

### 👋 نظام الترحيب
- رسالة ترحيب تلقائية لأي عضو جديد
- منح رول تلقائي (auto-role)

### 📊 نظام الخبرة والمستويات
- كسب الخبرة بالرسائل
- صعود المستويات تلقائياً
- إشعارات صعود المستوى

### 🔴 كشف البث المباشر
- دعم Twitch, YouTube, TikTok, Kick
- إشعارات تلقائية عند بدء البث
- تكامل مع الموقع الإلكتروني

## 🚀 التثبيت المحلي

### 1. إنشاء البوت على Discord
1. اذهب إلى [Discord Developer Portal](https://discord.com/developers/applications)
2. أنشئ تطبيق جديد (New Application)
3. من القائمة الجانبية اختر **Bot**
4. فعّل **Privileged Gateway Intents**:
   - ✅ Message Content Intent
   - ✅ Server Members Intent
   - ✅ Presence Intent
5. انسخ **Token** البوت

### 2. إعداد الصلاحيات
1. من القائمة الجانبية اختر **OAuth2 → URL Generator**
2. اختر **Scopes**: `bot` و `applications.commands`
3. اختر **Bot Permissions**: `Administrator`
4. انسخ الرابط وافتحه في المتصفح لإضافة البوت لسيرفرك

### 3. إعداد MongoDB (اختياري لكن موصى به)
1. اذهب إلى [MongoDB Atlas](https://www.mongodb.com/atlas) وأنشئ حساب مجاني
2. أنشئ مشروع جديد → Build a Database → Choose Free (M0)
3. اختر المنطقة الأقرب لك
4. اضبط Database Access (أنشئ مستخدم)
5. اضبط Network Access (أضف `0.0.0.0/0`)
6. من **Database → Connect → Drivers** انسخ **Connection String**
7. استبدل `<password>` بكلمة مرور المستخدم

### 4. إعداد المشروع
```bash
# استنساخ المستودع
git clone https://github.com/msolimaneissa-cmd/discord-bot.git
cd discord-bot

# تثبيت المكتبات
npm install

# إنشاء ملف .env
cp .env.example .env
```

### 5. تعديل ملف .env
```env
# مطلوب
DISCORD_TOKEN=token_البوت_هنا
CLIENT_ID=معرف_التطبيق_هنا

# موصى به
MONGODB_URI=mongodb+srv://المستخدم:كلمة_المرور@cluster.mongodb.net/discord_bot

# اختياري
GUILD_ID=معرف_السيرفر_هنا
BOT_STATUS=إدارة السيرفر
BOT_ACTIVITY_TYPE=PLAYING
```

### 6. تشغيل البوت
```bash
npm start
```

## ☁️ النشر على Render.com (مجاني)

### الخطوة 1: إعداد المستودع
1. ارفع الكود إلى GitHub
2. تأكد أن `.env` مضاف في `.gitignore`

### الخطوة 2: إنشاء خدمة على Render
1. اذهب إلى [Render](https://render.com) وأنشئ حساب
2. اختر **New → Blueprint**
3. اربط مستودع GitHub الخاص بك
4. سيتم اكتشاف ملف `render.yaml` تلقائياً

### الخطوة 3: إضافة متغيرات البيئة
في Render Dashboard، أضف المتغيرات التالية في **Environment**:
- `DISCORD_TOKEN` - رمز البوت
- `CLIENT_ID` - معرف التطبيق
- `MONGODB_URI` - رابط قاعدة البيانات (اختياري)
- `GUILD_ID` - معرف السيرفر (اختياري)

### الخطوة 4: النشر
اضغط **Apply** وسيتم نشر البوت تلقائياً!

## 🔄 إعداد UptimeRobot للحفاظ على البوت حياً 24/7

### لماذا UptimeRobot؟
Render في الخطة المجانية قد يوقف البوت بعد فترة من عدم النشاط. UptimeRobot يرسل طلبات دورية للحفاظ على البوت نشطاً.

### الخطوات:
1. اذهب إلى [UptimeRobot](https://uptimerobot.com) وأنشئ حساب مجاني
2. اضغط **+ Add New Monitor**
3. الإعدادات:
   - **Monitor Type**: HTTP(s)
   - **Friendly Name**: Family Legends Bot
   - **URL**: `https://your-app-name.onrender.com/health`
   - **Monitoring Interval**: كل 5 دقائق
4. اضغط **Create Monitor**

### نقاط النهاية المتاحة:
- `/` - معلومات البوت الكاملة
- `/health` - للتحقق من الصحة (موصى به لـ UptimeRobot)
- `/ping` - اختبار الاتصال
